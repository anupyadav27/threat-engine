# IBM Cloud Compliance Python Engine

Enterprise-grade compliance scanning engine for IBM Cloud infrastructure.

## Quick Start

### Installation
```bash
pip install -r requirements.txt
```

### Configuration
```bash
export IBM_CLOUD_API_KEY="your-api-key"
export IBM_CLOUD_REGION="us-south"
```

### Run Scan
```bash
python run_engine.py
```
